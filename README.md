# React JS Project

## Installation
- Run this command to install all dependencies
`npm install`
- Run this commend to start reactjs application as development
`npm start`
- Access to http://localhost:3000 to verify


## Project Folder

**General Folders**
- `src/assets`: Contains all static files such as: images, fonts...
- `src/constants`: Contains all constants variables that uses throughout the project.
- `src/utils`: Contains all Helpers (for example: toStringFormat, parseURL, formatPrice...)
- `src/lang`: Contains all languages, locale files....
- `src/services`: Contains the services that call to Back End.

**React Folders**
- `src/components`: Contains all reuse components (ProductList, Header...)
- `src/containers`: Contains all pages of the website (Home Page, Product Page, Cart Page...)
- `src/hooks`: Contains all react hooks that will share the logic of the components
- `src/contexts`: Contains all the react context that will share the data in application (optional, will use redux in most cases)

**Redux, Redux-Saga Folders**
- `src/actions`: Contains all actions creators.
- `src/reducers`: Contains all reducers
- `src/sagas`: Contains all sagas.
